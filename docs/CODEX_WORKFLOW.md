# CODEX_WORKFLOW

## Rules
- Pick the top unchecked task from docs/TASKS.md
- Make the smallest safe change that advances that task
- Run tests after changes
- Commit in small units
- Update README if behavior changes
- Move completed tasks from In Progress / Ready to Done
- If blocked, write the reason in docs/BLOCKED.md
- Do not start broad refactors unless required by the current task
